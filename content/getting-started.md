# Getting Started With V3

## Prerequisties

First clone the repository down from github:

::code-group
  ```bash [git]
  git clone https://github.com/crbroughton/v3
  ```
::

Then CD into the folder you've just created and install the dependencies

::code-group
  ```bash [pnpm]
  cd v3 && pnpm i
  ```
::
